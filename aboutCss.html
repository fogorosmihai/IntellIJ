CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

CSS Syntax:

SELECTOR {property:value;}

CSS Selectors:

The element Selector:

  • The element selector selects elements based on the element name.




The ID Selector:

  • The id selector uses the id attribute of an
    HTML element to select a specific element
  • The id of an element should be unique
  • To select an element with a specific id,
    write a hash (#) character, followed by the id of the element


The Class Selector:

  • The class selector selects elements with a specific class attribute.
  • To select elements with a specific class, write a period (.)
    character, followed by the name of the class.
